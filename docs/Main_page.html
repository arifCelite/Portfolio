Welcome to Cybex

Cybex has been created to educate and help students and individuals that are interested in learning more information about cyber security. On these web pages, you will have the opportunity to learn about cryptography and brute force password breaking. I have created a series of different web pages with some interactive screens that will make understanding and learning exciting and interesting. At the end of each page, you will have the chance to take a small quiz, with some questions about the topic you have read about. Explore and have fun!!!
